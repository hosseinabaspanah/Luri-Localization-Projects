# Luri Localization Projects

This repository contains a collection of software localization projects translated and adapted into **Luri** by **Hossein Abaspanah**.  

The goal is to preserve, support, and modernize the Luri language by making digital tools accessible for Luri-speaking communities.

## 📌 About This Repository
This repository serves as an official reference for:
- Luri language localizations of apps, software, and digital interfaces
- Translation resources and terminology
- Documentation for each localization project
- Versioning, licensing, and contribution guidelines

Each project has its own folder with a dedicated README, assets (screenshots), and version notes.

## 🎯 Purpose
- Provide official access to Luri-localized software
- Maintain transparency about who created and maintains the localizations
- Allow developers or contributors to reuse or extend Luri translations
- Preserve linguistic standards across modern digital platforms

## 🛠 How to use
- Visit a project's folder to find the detailed README with contribution notes, links to translation platforms (Weblate/Crowdin), and screenshots.
- If you are a developer and want to include a Luri translation, contact me or open an issue/pull request.
