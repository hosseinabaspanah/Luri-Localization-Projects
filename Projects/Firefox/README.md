# Firefox — Luri Localization

Documentation of the Luri language localization for Mozilla Firefox (Android).

## Project Details

- Application: Mozilla Firefox
- Platform: Android, Windows
- Type: Open-source
- Localization: User Interface (UI)
- Language: Luri Bakhtiari
- First published in the public version: Android (139), Windows (published only on Nightly version)
- Role: Localization Manager / Translator

## Localization Platform

This localization was contributed through Mozilla’s official translation platform ([pontoon](https://pontoon.mozilla.org/)).  
Attribution and contribution records are available on the platform.

## Scope

- Menus and settings  
- Dialogs and system messages  
- Core interface elements  

## Status

- Completed
- Maintained alongside upstream updates when required
